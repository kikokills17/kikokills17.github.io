CORE 2 LAB

  • 1.CRAIGSLIST AD +
    • Recreate the Craigslist listing as a website with Glitch. Use as much HTML, CSS and/or Javascript you feel comfortable with. Feel free to interpret the design however you want.
    • https://kiko-lab-p1.glitch.me/
  • 2.GITHUB PAGE +
    • Create a personal website hosted on Github pages that will have links to all your assignments from studio and lab. Think of this as building an "online house" for yourself. How do you want visitors to feel when they visit your house? How would you decorate it? Consider how it can grow as you will be building upon this page throughout the semester.
    • https://kikokills17.github.io/
  • 3.EXPRESSION +
    • Choose a piece of written text at least 50 words long — song lyrics, a passage from a book, a short essay, or an interview. Design and build a website that expresses this dialogue. Experiment with different ways to style and interact with text. You may include images and other media, but the focus should be on typography.
    • https://kikokills17.github.io/Lab-expression/intro.html
  • 4.RESPONSIVE SCREENS +
    • Create a website that displays different content when viewed on desktop and mobile. The user should browse your site on both devices at the same time to fully experience it. Example: a website that displays images on mobile and text on desktop.
    • https://kikokills17.github.io/Responsive-Screens/
  • 5.CALENDAR +
    • Create a calendar using CSS grid. The calendar does not have to be a traditional calendar, feel free to think more abstractly. At least two dates should link to a new subpage. The content and style of the subpages is up to you. You can post writing, images, video, sound, etc.
    • https://lab-canlendar.glitch.me/
